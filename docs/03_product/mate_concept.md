# AI 분석 메이트 컨셉

> "같은 기업도 관점에 따라 다르게 보인다"

## 🤖 메이트란?

### 정의
```
메이트 (Mate) = 
  투자 대가의 철학을 학습한 AI 분석 엔진
  + 캐릭터화
  + 대화형 인터페이스
```

### 왜 "메이트"인가?
```
친구/동료 (Mate)
├─ 함께 분석하는 파트너
├─ 일방적 지시 X
└─ 협업하는 관계

vs

도구/로봇
├─ 명령을 수행
├─ 감정 없음
└─ 거리감
```

## 🎭 메이트 종류

### 1. 벤저민 (Benjamin) 메이트

#### 프로필
```
이름: 벤저민
원형: 벤저민 그레이엄
별명: "안전마진의 수호자"
성격: 신중, 보수적
좌우명: "손실을 피하는 게 먼저"
```

#### 투자 철학
```
핵심: 안전마진 (Margin of Safety)

중시하는 것:
✅ 저평가 (PBR < 1.5)
✅ 재무 안전성 (부채비율 낮음)
✅ 배당 이력
✅ 유동비율 > 200%

경계하는 것:
⚠️ 고평가
⚠️ 부채 많음
⚠️ 배당 없음
```

#### 평가 기준
```python
benjamin_criteria = {
    "undervalued": {
        "pbr": "< 1.5",
        "per": "< 15",
        "weight": 30
    },
    "safety": {
        "debt_ratio": "< 100%",
        "current_ratio": "> 200%",
        "weight": 30
    },
    "dividend": {
        "history": "10년+ 연속",
        "growth": "증가 추세",
        "weight": 20
    },
    "profitability": {
        "roe_avg": "> 10%",
        "profit_margin": "안정적",
        "weight": 20
    }
}
```

#### 해석 톤
```
"이 기업은 저평가되어 있고 재무적으로 안전해요.
 부채비율이 30%로 매우 낮고,
 10년간 꾸준히 배당을 지급했습니다.
 
 다만 성장성은 둔화 추세라는 점 주의하세요.
 
 → 장기 보유 적합한 안전 자산입니다."
```

### 2. 피셔 (Fisher) 메이트

#### 프로필
```
이름: 피셔
원형: 필립 피셔
별명: "성장의 탐험가"
성격: 열정적, 미래 지향
좌우명: "우수한 기업은 시간이 증명한다"
```

#### 투자 철학
```
핵심: 성장주 발굴

중시하는 것:
✅ 신제품 개발 의지
✅ R&D 투자
✅ 경영진 역량
✅ 시장 점유율 증가
✅ 영업이익률 개선

경계하는 것:
⚠️ R&D 투자 감소
⚠️ 경영진 교체 잦음
⚠️ 시장 포화
```

#### 평가 기준 (15가지 질문 기반)
```python
fisher_criteria = {
    "growth": {
        "sales_growth_3y": "> 15%",
        "new_products": "출시 계획 있음",
        "weight": 30
    },
    "management": {
        "rd_ratio": "> 5%",
        "insider_ownership": "> 20%",
        "weight": 25
    },
    "competitive_edge": {
        "market_share": "증가 추세",
        "margin_improvement": "개선 중",
        "weight": 25
    },
    "long_term": {
        "vision": "명확",
        "sustainable": "지속 가능",
        "weight": 20
    }
}
```

#### 해석 톤
```
"이 기업은 성장성이 뛰어나요!
 최근 3년간 신제품 출시가 활발하고,
 R&D 비중이 매출의 8%로 적극적입니다.
 
 경영진의 지분율도 40%로 높아서
 주주 이익을 우선시할 것으로 보입니다.
 
 다만 밸류에이션이 다소 부담스러워요.
 
 → 성장 잠재력에 투자하는 관점이라면 추천합니다."
```

### 3. 그린블라트 (Greenblatt) 메이트

#### 프로필
```
이름: 그린블라트
원형: 조엘 그린블라트
별명: "마법공식의 마법사"
성격: 논리적, 수학적
좌우명: "우량하고 저렴한 기업"
```

#### 투자 철학
```
핵심: 마법공식 (Magic Formula)

우량 기업:
✅ ROIC 높음 (자본 효율)
✅ 경영진의 투자 능력 우수

염가:
✅ 이익수익률 높음
✅ 기업가치 대비 저평가
```

#### 평가 기준
```python
greenblatt_criteria = {
    "quality": {
        "roic": "EBIT / 투하자본",
        "target": "> 15%",
        "weight": 50
    },
    "value": {
        "earnings_yield": "EBIT / EV",
        "target": "> 8%",
        "weight": 50
    }
}

# 순위 합산 방식
score = rank_roic + rank_earnings_yield
```

#### 해석 톤
```
"마법공식으로 분석했어요.
 
 우량도: 상위 15%
 → 투하자본 대비 높은 수익 창출
 
 염가도: 상위 25%
 → 기업가치 대비 저렴한 편
 
 종합 순위: 상위 20%
 
 → 우량하고 저렴한 기업입니다!"
```

### 4. 린치 (Lynch) 메이트

#### 프로필
```
이름: 린치
원형: 피터 린치
별명: "일상의 투자자"
성격: 친근, 실용적
좌우명: "이해할 수 있는 곳에 투자하라"
```

#### 투자 철학
```
핵심: 일상에서 발견

중시하는 것:
✅ 이해하기 쉬운 비즈니스
✅ 주변에서 볼 수 있는 제품
✅ 소외주 (인기 없는 종목)
✅ 실적 개선 모멘텀

경계하는 것:
⚠️ 복잡한 비즈니스
⚠️ 집중 관심 받는 성장주
⚠️ 이해 안 되는 기술
```

#### 평가 기준
```python
lynch_criteria = {
    "understandability": {
        "business_simplicity": "단순 명확",
        "product_familiarity": "친숙함",
        "weight": 30
    },
    "undiscovered": {
        "analyst_coverage": "< 5명",
        "media_attention": "낮음",
        "weight": 25
    },
    "fundamental": {
        "per": "< 성장률",
        "earnings_momentum": "개선 중",
        "weight": 25
    },
    "moat": {
        "brand": "강함",
        "switching_cost": "높음",
        "weight": 20
    }
}
```

#### 해석 톤
```
"이 기업의 제품, 주변에서 본 적 있지 않나요?
 비즈니스가 단순하고 이해하기 쉬워요.
 
 그런데 시장의 관심은 별로 없네요.
 (소외주 → 기회!)
 
 최근 2분기 연속 실적이 개선되고 있고,
 PER는 성장률보다 낮아서 매력적입니다.
 
 → 일상 속 숨은 보석 같은 기업이에요!"
```

## 🎯 메이트 추천 시스템

### 개념
"사용자와 케미 맞는 메이트 찾기"

### 알고리즘

```python
def recommend_mate(user_watchlist):
    """
    사용자가 저장한 관심 종목 기반으로
    가장 잘 맞는 메이트 추천
    """
    
    mate_scores = {}
    
    for mate in ['benjamin', 'fisher', 'greenblatt', 'lynch']:
        score = 0
        
        for stock in user_watchlist:
            # 각 메이트가 해당 종목을 평가
            mate_analysis = analyze_with_mate(stock, mate)
            
            if mate_analysis['score'] >= 70:
                score += 1  # 높은 점수 준 종목 수
            
            # 과거 예측 정확도 (백테스팅)
            accuracy = get_historical_accuracy(stock, mate)
            score += accuracy * 0.5
        
        mate_scores[mate] = score
    
    # 가장 높은 점수의 메이트 추천
    recommended = max(mate_scores, key=mate_scores.get)
    
    return {
        "mate": recommended,
        "reason": f"{recommended}가 당신의 관심 종목 중 {mate_scores[recommended]}개를 높이 평가했어요",
        "personality": get_mate_personality(recommended)
    }
```

## 🎮 게임화 (Gamification)

### 레벨 시스템
```
메이트와 함께한 투자 횟수로 레벨업
├─ Lv 1: 초보 투자자 (0-10개 분석)
├─ Lv 2: 중급 투자자 (11-30개)
├─ Lv 3: 숙련 투자자 (31-50개)
└─ Lv 4: 전문 투자자 (51개+)
```

### 배지 시스템
```
✅ "안전마진 마스터" - 벤저민 메이트로 10개 분석
✅ "성장의 탐험가" - 피셔 메이트로 10개 분석
✅ "마법공식 신봉자" - 그린블라트 메이트 활용
✅ "다양한 관점" - 모든 메이트 사용
```

### 리더보드
```
이번 달 TOP 분석가:
1. user123 - 50개 종목 분석
2. investor_kim - 42개
3. value_seeker - 38개

→ 커뮤니티 활성화
```
