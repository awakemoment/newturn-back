## 🏠 **메인 페이지**

### **Top Picks**
- 4개 메이트가 가장 높이 평가한 종목 표시
- 클릭하면 종목 상세로 이동
- 실시간 업데이트

### **검색**
- 종목명 또는 티커로 검색
- 2글자 이상 입력
- 자동완성 지원

## 📊 **스크리닝 페이지**

### **뷰 모드**

#### **📊 테이블 뷰** (추천)
```
✅ 모든 메이트 점수를 한눈에 비교
✅ 3,912개 종목 전체 탐색
✅ 빠른 정렬/필터링
```

**사용법:**
1. 정렬 버튼 클릭 (평균, 베니, 그로우, 매직, 데일리)
2. 종목 클릭 → 상세 분석
3. "비교함" 버튼 → 종목 비교

#### **🎴 카드 뷰**
```
✅ 메이트별 추천 종목
✅ 시각적으로 편안함
✅ 큐레이션 탭
```

**사용법:**
1. 메이트 탭 선택 (🎩 베니, 🌱 그로우, 🔮 매직, 🎯 데일리)
2. 각 메이트가 추천하는 Top 12 종목 표시
3. 카드에서 4개 메이트 점수 확인

### **필터링 예시**

**"모든 메이트가 80점 이상인 종목"**
```
테이블 뷰 → URL에 직접:
/screen?min_all_mates=80
```

**"평균 90점 이상"**
```
/screen?min_avg_score=90
```

## 🔍 **종목 상세 페이지**

### **핵심 지표 (상단)**
```
💰 TTM FCF: 최근 12개월 잉여현금흐름
📊 FCF Margin: FCF / Revenue
📈 ROE: 자기자본이익률
⚖️ Debt Ratio: 부채비율
```

### **현금흐름 차트**
```
📈 20분기 트렌드
✅ 왼쪽→오른쪽 시간 흐름
💡 OCF, FCF, Revenue 동시 표시
```

### **투자 메이트 분석**

#### **🎩 베니 (Benny)**
- **철학**: 안전마진
- **중시**: 부채비율, 유동비율, FCF 안정성
- **추천**: 70점 이상 = 장기 투자 적합

#### **🌱 그로우 (Grow)**
- **철학**: 성장주
- **중시**: 매출 성장률, ROE, FCF 성장
- **추천**: 70점 이상 = 성장 잠재력 우수

#### **🔮 매직 (Magic)**
- **철학**: 마법공식 (우량+저렴)
- **중시**: ROE, FCF 마진
- **추천**: 70점 이상 = 마법공식 후보

#### **🎯 데일리 (Daily)**
- **철학**: 일상 속 발견
- **중시**: 이해도, 실적 모멘텀
- **추천**: 70점 이상 = 성장 모멘텀

### **10-K 인사이트** (Standard 티어+)
```
📄 제품별 매출 (iPhone, Mac, iPad 등)
🌍 지역별 매출 (Americas, Europe, China 등)
🚨 신규 리스크
📊 주요 변화
💰 핵심 비율 (Gross/Operating/Net Margin)
```

## 🎯 **포트폴리오 관리**

### **추가**
1. 종목 상세 → "포트폴리오에 추가"
2. 매수가, 수량 입력
3. 저장

### **Hold/Sell 시그널**
```
🟢 Hold: 메이트 점수 70점 이상 유지
🟡 주의: 점수 하락 (-5점 이상)
🔴 Sell: 점수 60점 미만
```

**판단 기준:**
- FCF 트렌드 (3분기 연속 하락 → 주의)
- 메이트 점수 변화
- 리스크 발견 (10-K)

## 🎓 **투자 철학 가이드**

### **🎩 벤저민 그레이엄 (Benjamin Graham)**
- **저서**: 《현명한 투자자》
- **핵심**: 안전마진 (Margin of Safety)
- **전략**: 저평가 + 안정성
- **적합**: 보수적 투자자

### **🌱 필립 피셔 (Philip Fisher)**
- **저서**: 《위대한 기업에 투자하라》
- **핵심**: 성장 잠재력
- **전략**: 우수 경영 + 혁신
- **적합**: 장기 성장주 투자자

### **🔮 조엘 그린블라트 (Joel Greenblatt)**
- **저서**: 《주식시장을 이기는 작은 책》
- **핵심**: 마법공식 (우량+저렴)
- **전략**: ROIC + Earnings Yield
- **적합**: 시스템 투자자

### **🎯 피터 린치 (Peter Lynch)**
- **저서**: 《전설로 떠나는 월가의 영웅》
- **핵심**: 일상에서 발견
- **전략**: 이해 가능한 기업
- **적합**: 실용적 투자자

## ❓ **FAQ**

### **Q: 메이트 점수는 어떻게 계산되나요?**
A: 재무 지표 기반 규칙 엔진입니다.
- 베니: 부채비율, 유동비율, FCF 안정성
- 그로우: 매출/FCF 성장률, ROE
- 매직: ROE, FCF 마진
- 데일리: 이해도, 실적 모멘텀

### **Q: 10-K 인사이트는 무엇인가요?**
A: SEC에 제출된 사업보고서(10-K)를 AI가 자동 분석한 결과입니다.
- 제품별/지역별 매출
- 신규 리스크 발견
- 주요 변화 추적

### **Q: 데이터는 얼마나 자주 업데이트되나요?**
A: 
- 재무 데이터: 분기별 (10-Q 공시 후 1일 이내)
- 메이트 점수: 재무 데이터 업데이트 시 자동
- 10-K 인사이트: 연 1회 (10-K 공시 후)

### **Q: 주가 데이터는 실시간인가요?**
A: Polygon.io 무료 플랜은 15분 지연입니다.

### **Q: 한국 주식도 지원하나요?**
A: 현재는 미국 주식만 지원합니다. 한국 주식은 추후 추가 예정입니다.

## 🎯 **추천 워크플로우**

### **새 종목 발굴**
```
1. 스크리닝 테이블 뷰
2. "평균 80점 이상" 정렬
3. 상위 20개 종목 확인
4. 관심 종목 비교함에 담기
5. 비교 페이지에서 상세 비교
6. 최종 선택 → 포트폴리오 추가
```

### **보유 종목 관리**
```
1. 포트폴리오 페이지
2. Hold/Sell 시그널 확인
3. 시그널 발생 시:
   - 종목 상세 재확인
   - 메이트 점수 변화 추적
   - 10-K 신규 리스크 확인
4. 판단 후 매도/보유 결정
```
