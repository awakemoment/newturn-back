## 🎯 **핵심 기능**

### **1. 스크리닝 시스템**
- 📊 **테이블 뷰**: 모든 메이트 점수 한눈에 비교
- 🎴 **카드 뷰**: 메이트별 추천 종목 큐레이션
- 🔍 **필터링**: 평균 점수, 개별 메이트 점수
- 📈 **정렬**: 5가지 정렬 옵션

### **2. 투자 메이트 시스템**
- 🎩 **베니 (Benjamin)**: 안전마진, 저평가
- 🌱 **그로우 (Fisher)**: 성장주, R&D
- 🔮 **매직 (Greenblatt)**: 마법공식, ROIC
- 🎯 **데일리 (Lynch)**: 일상 발견, 이해도

### **3. 종목 상세 분석**
- 💰 **재무 지표**: TTM 기반 핵심 지표
- 📈 **현금흐름 차트**: 20분기 트렌드
- 🤖 **AI 인사이트**: 4개 메이트 종합 분석
- 📄 **10-K 인사이트**: 제품/지역별 매출, 신규 리스크

### **4. 포트폴리오 관리**
- 📊 **포트폴리오 생성/관리**
- 🎯 **Hold/Sell 시그널**
- 📈 **성과 추적**

## 🛠️ **기술 스택**

### **백엔드**
- Django 5.0
- Django REST Framework
- PostgreSQL (권장) / SQLite
- Celery (백그라운드 작업)
- Stripe (결제)

### **프론트엔드**
- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

### **데이터 소스**
- SEC EDGAR API (재무 데이터)
- Polygon.io (주가 데이터)
- 자체 10-K 파서 (정성 데이터)

## 📦 **설치 및 실행**

### **백엔드**
```bash
cd business/newturn-back

# 환경 설정
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt

# DB 마이그레이션
python manage.py migrate

# 서버 실행
python manage.py runserver
```

### **프론트엔드**
```bash
cd business/newturn-front

# 의존성 설치
npm install

# 개발 서버
npm run dev

# 프로덕션 빌드
npm run build
```
