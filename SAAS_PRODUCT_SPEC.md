## 🎯 제품 개요

### **핵심 가치 제안:**
```
"5분이면 워렌 버핏처럼 기업을 분석할 수 있습니다"
```

### **타겟 사용자:**
1. **미국 주식 개인 투자자** (메인)
2. **가치투자 학습자**
3. **재무 분석 학생**
4. **소규모 투자 클럽**

### **핵심 차별점:**
```
기존 도구:
- Finviz: 스크리너만, 밸류에이션 없음
- Seeking Alpha: 뉴스 위주, 계산 없음
- TradingView: 차트 위주, 펀더멘털 약함

Newturn:
✅ 4개 메이트 (4가지 관점)

✅ 교육적 설명 (학습)
✅ 컨센서스 (의사결정 도움)
```

## 🎯 MVP (Minimum Viable Product)

### **Phase 0A (Week 1-8): 핵심 기능만**

#### **기능 1: 종목 검색**
```
┌─────────────────────────────────────┐
│ 🔍 종목 검색                         │
│ [____________] [검색]                │
│                                     │
│ 예: NVDA, AAPL, TSLA                │
└─────────────────────────────────────┘

기술:
- Frontend: Autocomplete (Ticker + Name)
- Backend: Stock.objects.filter(ticker__icontains=query)
- 데이터: 기존 5,036개 종목
```

```

│ [상세 →]                             │
│                                     │
│ 🌱 그로우 (Fisher)                  │
│ 적정가: $380.50                      │
│ 괴리율: +32.7% 📉 고평가             │

│                                     │
```
